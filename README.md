Smart Parking Dornbirn is a three-part repository for the [Smart City Dornbirn](https://smartcitydornbirn.com/) competition, consisting of an arduino project, **a website** and, if we have enough time, an app. 

## Requirements

- NodeJS
- NPM

## Installation

```
npm run setup
npm start
```
